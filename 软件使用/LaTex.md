#### 插入多行公式时，空格问题

第一个正确，第二个不对，因此在 **插入公式时，就不要有空格** ，避免不必要的麻烦。

```latex
\begin{equation}
\left\{ \begin{array}{l}
	r_k=\sum_{j=1}^p{w_{kj}x_j\text{,}}\\
	v_k=r_k-\theta _k,\\
	y_k=\varPhi \left( v_k \right),\\
\end{array} \right.	
\end{equation}
```

```latex
  \let\ps@plain\ps@empty

    \fancyhf{} %
    \fancyhead[LO,RE]{...} %
    \fancyhead[RO,LE]{...} %
    \fancyfoot[C]{...} %
    \renewcommand{\headrulewidth}{0.4pt}  %

}
```
